# Tencent Xinge Push

A simple Composer port for the [Tencent Xinge Push](http://xg.qq.com) PHP SDK.

- [Official Documentation](http://developer.qq.com/wiki/xg/)
- [Usage Samples](docs)

## Installation

```sh
composer require javareact/xinge
```
